# 商品分类小程序

这是一个完整的微信小程序项目，包含商品分类、社区分享、收藏管理等功能。

## 功能特性

### 🛍️ 商品分类页面
- 左侧分类导航，支持切换不同商品分类
- 右侧商品列表展示，包含商品图片、标题、价格等信息
- 商品详情弹窗，支持选择规格和数量
- 搜索功能，支持关键词搜索商品
- 购物车功能，支持添加商品到购物车

### 👥 社区页面
- 用户动态展示
- 点赞和评论功能
- 发布动态功能
- 图片分享功能

### ❤️ 收藏页面
- 收藏商品管理
- 批量操作（全选、删除、加入购物车）
- 收藏统计信息

### 📸 图片分享页面
- 图片轮播展示
- 点赞、收藏、下载功能
- 相关推荐

## 快速开始

### 1. 导入项目
1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择项目根目录
4. 输入你的AppID（如果没有可以选择"测试号"）

### 2. 配置项目
1. 在 `project.config.json` 中修改 `appid` 为你的小程序AppID
2. 在 `app.json` 中根据需要调整页面配置

### 3. 运行项目
1. 点击微信开发者工具的"编译"按钮
2. 在模拟器中预览效果
3. 使用真机调试功能在手机上测试

## 主要功能说明

### 商品分类功能
- 支持多个商品分类切换

- 商品详情弹窗，支持规格选择和数量调整
- 搜索功能，支持关键词搜索

## 技术特点

- 使用微信小程序原生开发
- 响应式设计，适配不同屏幕尺寸
- 模块化代码结构，便于维护
- 统一的UI设计风格
- 完整的用户交互体验

## 注意事项

1. 本项目使用的是模拟数据，实际使用时需要连接真实的后端API
2. 图片资源使用的是占位图片，实际使用时需要替换为真实的商品图片
3. 购物车数据目前存储在本地，实际使用时建议同步到服务器
4. 支付功能为模拟实现，实际使用时需要接入真实的支付接口
